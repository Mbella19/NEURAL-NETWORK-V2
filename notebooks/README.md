# Notebooks

This directory is reserved for Jupyter notebooks used for:

- **Data Exploration**: Analyzing EURUSD price data and market patterns
- **Feature Analysis**: Visualizing technical indicators and their distributions
- **Model Debugging**: Inspecting Analyst predictions and Agent behavior
- **Backtest Analysis**: Detailed examination of trading performance

## Suggested Notebooks

1. `exploration.ipynb` - Initial data exploration and visualization
2. `feature_engineering.ipynb` - Feature correlation and importance analysis
3. `analyst_evaluation.ipynb` - Market Analyst prediction quality
4. `agent_behavior.ipynb` - Agent action distribution and trade analysis
5. `backtest_analysis.ipynb` - Detailed backtest metrics and equity curves

## Usage

```bash
# Start Jupyter
cd notebooks
jupyter notebook

# Or use JupyterLab
jupyter lab
```
